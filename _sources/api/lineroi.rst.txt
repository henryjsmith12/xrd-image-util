=======
LineROI
=======

:mod:`xrdimageutil.roi.LineROI`
~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. currentmodule:: xrdimageutil.roi.LineROI

The ``LineROI`` class provides a line segment region of interest that can be applied to
a 3D dataset. The region is defined by the pixels that the line segment intersects 
(see Bresenham's algorithm).

Attributes
^^^^^^^^^^

.. py:attribute:: endpoints
    :type: dict

    Coordinate endpoints for line segment.
    
.. py:attribute:: calculation
    :type: dict

    Output calculation and the dims to calculate across.
